# My Neovim settings

## Installation:
1. Install Neovim
2. Install Neovim-plug
3. Install Golang(required by hexokinase)
4. Clon to home/user/.config/
> $ git clone https://github.com/thyrgrim/vimrc
6. Start Neovim and install plugins with:
> :PlugInstall
